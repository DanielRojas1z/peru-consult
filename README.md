## Install
Usando composer desde [packagist.org](https://packagist.org/packages/giansalex/peru-consult)
```bash
composer require giansalex/peru-consult
```

### Requerimientos
- PHP 7.1 o superior.

### Servicios Disponibles
- Consulta de Dni
- Consulta de Ruc
- Validez de usuario SOL
