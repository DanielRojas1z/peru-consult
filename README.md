Consultas de DNI y RUC sin uso de captcha ni OCR.

# Install
Using composer from [packagist.org](https://packagist.org/packages/giansalex/peru-consult)
```bash
composer require giansalex/peru-consult
```

### Disponibles
- Ruc (SUNAT).
- Dni (RENIEC).

### Requirements
- Ruc requier extension `dom`.
- Dni require extensiones `gd`.

### API
Puede utilizar el API REST [peru-consult-api](https://github.com/giansalex/peru-consult-api)
